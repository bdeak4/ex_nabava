# ExNabava

ExNabava is unofficial elixir lib for accessing nabava.net public data.

## Installation

```elixir
def deps do
  [
    {:ex_nabava, "~> 0.3.2"}
  ]
end
```

add `device_id` in [config.exs](config/config.exs)
